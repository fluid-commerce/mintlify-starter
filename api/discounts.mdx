The Discounts API lets you create, update, and manage promotional discount codes for your Fluid storefront. Use it to run seasonal sales, reward loyal customers, and configure usage limits and expiration dates. Discounts can be applied to carts via the Carts API and support percentage off, fixed amount off, and free shipping discount types.

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/v2025-06/discounts` | List discounts |
| `POST` | `/api/v2025-06/discounts` | Create a discount |
| `GET` | `/api/v2025-06/discounts/{id}` | Get a discount |
| `PUT` | `/api/v2025-06/discounts/{id}` | Update a discount |
| `DELETE` | `/api/v2025-06/discounts/{id}` | Delete a discount |
| `POST` | `/api/v2025-06/discounts/{id}/clone` | Clone a discount |
| `DELETE` | `/api/v2025-06/discounts/bulk_delete` | Bulk delete discounts |

## Create a Discount

Send a `POST` request with a JSON body to create a new discount code. Specify the `discount_type` to control how the discount is calculated, and set `usage_limit` to cap how many times the code can be redeemed.

```bash
curl -X POST https://api.fluid.app/api/v2025-06/discounts \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "SUMMER20",
    "discount_type": "percentage",
    "value": "20.0",
    "usage_limit": 500,
    "active": true,
    "expires_at": "2025-08-31T23:59:59Z"
  }'
```

<ParamField body="code" type="string" required>
  The discount code string customers enter at checkout. Must be unique within your account. Case-insensitive.
</ParamField>

<ParamField body="discount_type" type="string" required>
  Type of discount to apply. Accepted values: `percentage`, `fixed_amount`, `free_shipping`.
</ParamField>

<ParamField body="value" type="string" required>
  Discount value. For `percentage`, provide a number between `0` and `100`. For `fixed_amount`, provide the monetary value to deduct. Not required for `free_shipping`.
</ParamField>

<ParamField body="usage_limit" type="integer">
  Maximum number of times this code can be redeemed across all customers. Omit for unlimited usage.
</ParamField>

<ParamField body="active" type="boolean">
  Whether the discount is active and can be applied to carts. Defaults to `true`.
</ParamField>

<ParamField body="expires_at" type="string">
  ISO 8601 timestamp after which the discount code is no longer valid. Omit for a non-expiring discount.
</ParamField>

## Discount Fields

<ResponseField name="id" type="integer">
  Unique numeric identifier for the discount.
</ResponseField>

<ResponseField name="code" type="string">
  The discount code string as entered by customers.
</ResponseField>

<ResponseField name="discount_type" type="string">
  The type of discount: `percentage`, `fixed_amount`, or `free_shipping`.
</ResponseField>

<ResponseField name="value" type="string">
  The discount value. Represents a percentage or monetary amount depending on `discount_type`.
</ResponseField>

<ResponseField name="usage_limit" type="integer or null">
  Maximum number of allowed redemptions, or `null` for unlimited.
</ResponseField>

<ResponseField name="times_used" type="integer">
  How many times this discount code has been successfully redeemed.
</ResponseField>

<ResponseField name="active" type="boolean">
  `true` if the discount is currently active and can be applied to carts.
</ResponseField>

<ResponseField name="expires_at" type="string or null">
  ISO 8601 expiration timestamp, or `null` if the discount has no expiry.
</ResponseField>
